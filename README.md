# 使用指南
## 安装
```
npm i @waylonzheng/vite-uglify-html-plugin -D
```
## 使用
```js
// vite.config.js
import uglifyHtmlPlugin from '@waylonzheng/vite-uglify-html-plugin'; // 这里我已经发npm包了，直接引
plugins: [uglifyHtmlPlugin()];
```
